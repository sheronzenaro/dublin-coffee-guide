
# ☕ Dublin Coffee Guide

🌐 **Live site:** [sheronzenaro.github.io/dublin-coffee-guide](https://sheronzenaro.github.io/dublin-coffee-guide/)

A website I built to share some of my favourite coffee shops around Dublin. It started as a first-year project at university, but I ended up putting a lot of myself into it — the cafés are all places I actually visited, the photos are mine, and the reviews are genuine.

---

## About

I wanted to build something that felt real rather than just a practice project. Dublin has a great coffee scene and I was already exploring it, so it made sense to document it this way. The site has grown into something I'm genuinely proud of.

---

## Features

- Multi-page layout — Home, Coffee Shops, Locations, Next Coffee Shops, and Contact
- Fully responsive — works on desktop, tablet, and mobile
- Image lightbox — click any photo to view it full screen
- Google Maps with all café locations
- Contact form with validation and a personalised confirmation page
- Real-time character counter on the message field
- Back to top button with smooth scroll
- RSS feed pulling in the latest coffee news
- SEO meta tags on all pages
- Custom favicon

---

## Built With

- HTML5
- CSS3 (Flexbox, Media Queries, Animations)
- JavaScript / jQuery
- Google Fonts — Playfair Display, Cormorant Garamond, Inter
- Font Awesome
- Google Maps Embed API
- GitHub Pages

---

## Pages

| Page | Description |
|------|-------------|
| Home | Introduction and overview |
| Coffee Shops | Reviews and photos of visited cafés |
| Locations | Map with all café addresses |
| Next Coffee Shops | Places on my list, plus a live coffee news feed |
| Contact | Form to send recommendations or get in touch |

---

## What I Learned

This was my first time building a complete multi-page website from scratch. Some things that stood out during the process:

- Getting the layout to work well across different screen sizes took more CSS than I expected
- Integrating the Google Maps embed and RSS feed was a good introduction to working with external APIs
- Deploying on GitHub Pages made the whole thing feel real in a way that running it locally didn't
